# joke-tag-api

API for tags management (CRUD)

## Clone

```bash
git clone https://github.com/SalathielGenese/joke-tag-api
```

## Install

```bash
cd joke-tag-api
```

```bash
yarn install
```

## Run app

```bash
yarn start
```

## Run tests

```bash
yarn test
```

## Run dev

Restart test and app on change

```bash
yarn dev
```

Restart test on change

```bash
yarn dev:test
```

Restart app on change

```bash
yarn dev:start
```
